DRA-5C Console

A workspace that walks the four-step data-readiness assessment end to end and assembles the 5C scorecard report.

How this works. Generation is done by the dra-5c agent, not this page — each step gives you a prompt to paste into the agent (or any Claude), and you paste the result back here. This page makes no API calls; your work autosaves locally in your browser and can be exported as a file.
