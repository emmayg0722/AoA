## This layer answers

**Where does the agent create business value?**

| Business Area | Agent Use Case |
|---|---|
| ERP | Order lookup, invoice explanation, master data assistant |
| CRM | Customer summary, opportunity insights |
| Finance | Reconciliation assistant, variance explanation |
| Supply chain | Inventory exception analysis |
| HR | Policy Q&A, onboarding assistant |
| IT support | Ticket triage, solution suggestion |
| Consulting | Proposal generation, project documentation |
| Data migration | Mapping assistant, validation assistant |
| Compliance | Regulation mapping, audit support |

This layer is where "AI demo" becomes "business solution."
