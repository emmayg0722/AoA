# Layer 18 — Agent Types Layer

You can also classify agents by role.

## Common agent types

| Agent Type | Purpose |
|---|---|
| Q&A Agent | Answers from knowledge |
| Research Agent | Searches, compares, summarizes |
| Workflow Agent | Executes process steps |
| Coding Agent | Writes/edits code |
| Data Agent | Queries and explains data |
| Document Agent | Generates structured documents |
| Support Agent | Handles tickets/questions |
| Sales Agent | Helps with leads/proposals |
| Compliance Agent | Checks against policies/regulations |
| Monitoring Agent | Watches events and alerts user |
| Personal Assistant Agent | Calendar/email/task support |
| Multi-agent System | Several agents collaborate |

For a typical AI-architecture consulting engagement, the most relevant ones are usually:

- Document Agent
- ERP Data Agent
- Internal Knowledge Agent
- Compliance Agent
- Workflow Agent
- AI Consultant Copilot
