# Layer 16 — Workflow and Process Layer

This is where architecture becomes operational.

A production agent needs a defined lifecycle:

1. User request
2. Intent detection
3. Permission check
4. Retrieve context
5. Plan task
6. Execute tools
7. Validate output
8. Ask human if needed
9. Final response/action
10. Log result
11. Evaluate run
12. Improve prompt/data/tooling

This is the real agentic loop.

The stack diagram shows the tools. But the architect must design the loop.
